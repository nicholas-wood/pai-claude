## Operating procedure quick-reference

**To deploy IRB:**
1. Roll rack out through shed door
2. Position with rear toward unload area
3. Drop the stabilizer leg
4. Release both slide latches
5. Push sled assembly rearward (~820 mm) until drop-in pivot pins engage
6. Confirm IRB bow strap is secured
7. Operate winch — let cable out, sled tilts down
8. Lift IRB from sled at chosen tilt

**To stow:** Reverse the above.
