## Summary

South Melbourne Lifesaving Club (SMLSC) facilities improvement project. Covers a clubhouse master plan, a board proposal for a single coordinated Clubhouse Renewal Project funded via VESEP (2:1 government:club), and a mechanical design task for custom mobile rescue board racks (GS-03).
